📱 Responsive UI

🎬 Fetch Movies and Tv Show

🔎 Search for Actors and Movies

🎥 Watch Trailers

## Setup .env file
```
PORT=5000
MONGO_URI=your_mongo_uri
NODE_ENV=development
JWT_SECRET=your_jwt_secre
TMDB_API_KEY=your_tmdb_api_key
```
## Run this app locally
```
npm run build
```
## Start the app
```
npm run start
```
